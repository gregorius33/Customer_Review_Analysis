# 고객 리뷰 분석 (GUI + ChatGPT 보고서)

엑셀 파일을 선택하면 리뷰 열을 자동/수동으로 매핑하고, ChatGPT API로 분석 보고서를 생성하는 GUI 프로그램입니다.

## 설치

```bash
pip install -r requirements.txt
```

## API 키 설정

1. 프로젝트 폴더에 `.env` 파일을 만듭니다.
2. [OpenAI API 키](https://platform.openai.com/api-keys)를 발급받아 아래처럼 입력합니다.

```
OPENAI_API_KEY=sk-xxxxxxxxxxxxxxxx
```

`.env.example`을 복사해 `.env`로 이름을 바꾼 뒤 키만 입력해도 됩니다.

## 실행

```bash
python main.py
```

1. **엑셀 파일 선택**으로 `.xlsx` 파일을 고릅니다.
2. **열 매핑**이 자동으로 시도됩니다. 리뷰 내용, 평점, 제품, 연령, 구매일자, 성별 등이 비어 있으면 드롭다운에서 해당 열을 선택합니다.
3. **분석 보고서 생성**을 누르면 "생성 중..." 상태가 표시되고, ChatGPT(gpt-4o-mini)로 보고서가 생성됩니다. 완료 시 "생성 완료."로 바뀝니다.
4. **보고서 저장**으로 마크다운(.md) 또는 텍스트 파일로 저장할 수 있습니다. 기본 저장 형식은 마크다운(.md)이며, 제목·표·목록이 마크다운 문법으로 저장됩니다.

보고서에는 기본 통계, 평균 평점, 평점별 분포, 긍정/부정/중립 리뷰 분포, 연령대·성별·구매일자 분포(데이터가 있을 때), 제품별 현황, 상세 분석, 개선점이 포함됩니다.

## 참고

- 계획/설계 문서: `.cursor/plans/` 폴더
- 리뷰 열이 다른 위치에 있어도 헤더 이름으로 매칭하거나, 화면에서 열을 선택해 사용할 수 있습니다.
